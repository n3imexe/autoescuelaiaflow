# AutoBot AI - Landing Page para Autoescuelas

Una landing page moderna y profesional para un chatbot de IA automatizado diseñado específicamente para autoescuelas.

## 🚀 Características

- **Diseño Moderno**: Interfaz limpia y profesional con TailwindCSS
- **Totalmente Responsiva**: Funciona perfectamente en todos los dispositivos
- **Interactiva**: Animaciones suaves y elementos interactivos
- **Demo de Chat**: Simulación interactiva del chatbot
- **Formulario de Contacto**: Validación de formulario en tiempo real
- **Optimizada para SEO**: Estructura semántica y metaetiquetas
- **Alto Rendimiento**: Código optimizado y carga rápida

## 📁 Estructura del Proyecto

```
windsurf-project/
├── index.html          # Página principal
├── styles.css          # Estilos personalizados
├── script.js           # Funcionalidad JavaScript
└── README.md           # Este archivo
```

## 🛠 Tecnologías Utilizadas

- **HTML5**: Estructura semántica moderna
- **TailwindCSS**: Framework CSS para diseño rápido
- **Font Awesome**: Iconos de alta calidad
- **JavaScript Vanilla**: Interactividad sin dependencias

## 🚀 Despliegue

### Para desarrollo local
1. Clona el repositorio
2. Abre `index.html` en tu navegador
3. No requiere servidor local

### Para producción
1. Sube los archivos a tu servidor web
2. Asegúrate de que todos los archivos estén en el mismo directorio
3. Configura tu dominio y SSL
